# Welcome to HTML-Markdown Converter

This is a **sample Markdown file** to demonstrate the conversion capabilities.

## Features

- Convert HTML to Markdown
- Convert Markdown to HTML
- CLI and VS Code extension

## Code Example

```javascript
const converter = require('html-md-converter');
const html = converter.markdownToHtml('# Hello');
console.log(html);
```

For more information, visit [our repository](https://github.com/jasdeepkhalsa/html-md-converter).
