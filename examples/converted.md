# Welcome to HTML-Markdown Converter

This is a **sample HTML file** to demonstrate the conversion capabilities.

## Features

*   Convert HTML to Markdown
*   Convert Markdown to HTML
*   CLI and VS Code extension

## Code Example

```
const converter = require('html-md-converter');
const markdown = converter.htmlToMarkdown('<h1>Hello</h1>');
console.log(markdown);
```
